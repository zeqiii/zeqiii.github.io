Install and Switch Between Multiple Java Environments

Compiling android source code, install JBoss, and so on, all need different Java environments like jdk6, jdk7, openjdk…

Can we install all these environments and switch to one version on the need?

Yes, by using command sudo update-alternatives --install, we can achieve this goal.
